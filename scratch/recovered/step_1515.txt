3: This plan outlines the design and implementation steps for **Phase 3.5 — Drag & Drop** gesture control in GestureOS.
4: 
5: ## Proposed Design
6: 
7: ### 1. Drag & Drop Gesture State Machine
8: We will track the Pinch state (Thumb + Index distance <= threshold):
9: - **Pinch Detection**:
10:   - When pinch is detected, we start a timer to measure the hold duration.
11:   - If `hold_duration < drag_threshold_ms` (configurable, default `200ms`), cursor movement is temporarily frozen to avoid cursor drift / accidental drags during a simple click.
12:   - If the pinch is released *before* `drag_threshold_ms`, we dispatch a standard Left Click (`pyautogui.click()`).
13:   - If the pinch is held *beyond* `drag_threshold_ms`:
14:     - We transition to **DRAGGING** mode.
15:     - We execute `pyautogui.mouseDown()`.
16:     - We log `[DRAG] Drag Start`.
17:     - We increment `drag_counter`.
18: - **Maintain Drag**:
19:   - While pinch is held, cursor movement resumes, allowing the user to drag the item.
20:   - The HUD overlay shows `DRAGGING` visual feedback in green.
21: - **Pinch Release**:
22:   - When pinch distance exceeds threshold, we release the drag.
23:   - We execute `pyautogui.mouseUp()`.
24:   - We log `[DRAG] Drag End | Duration = X.XXs`.
25:   - Reset timers.
